# Duck Cards

  Task management app with best practices implementation of React, Redux, Node, MongoDB

## Requirements

- Node
- MongoDB
- Mongoose
- Express
- Webpack
- React
- Redux
- React-Redux

### Usage

From within the root directory:

```sh
npm install
npm run build
node Server/server.js
```
